## Install dependencies

```bash
npm install
```

## Run web server

```bash
npm start
```

## Lint and fix

```bash
npm run lint
```

## Run unit tests

```bash
npm test
# or
npm run test:unit
```

## Run E2E tests

```bash
npm run test:e2e
```

## Generate locales

```bash
npm run locales
```
